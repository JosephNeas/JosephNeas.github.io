Graphs of Colorado's top 5 biggest cities

This data was found here. Labels the cities of Colorado from the biggest population to the smallest from 2019.

Version 1

This is the standard graph from the Chart JS documentation. The only thing I did to change this is put in my data from the Colorado demographics site.

  • Here I changed the graph to be in a container to fit better on the screen of the website. I did this by putting class="container" in the canvas declaration <canvas class="container"> for the second graph.
  • Next thing I was add a title to the chart it self called "Version 2". I did this using the title syntax from Container JS while changing the display to be true, the title's font size to 30, and making the font color black.
  • Next, I made the colors of the bar chart a bit darker to make them stand out more by messing with the transparency of the RGBA values. I then wanted to make the user know when they are hovering over the different data. I did this by first changing the transparency of the bars to be solid when hovering over it. I next changed the border color to black and made the width a bit bigger.
